## Overview

I have been lucky enough to work with some great software developers during my PhD, who have taught me a lot. While I am still have a lot to learn, I have enjoyed writing documentation and learning more about the process of development. You can see some projects I've worked on below.

### Writing documentation for seaduck
![Salinity equation terms](/assets/images/seaduck_logo.png){: .align-center}
Seaduck is a software package designed by [Wenrui Jiang](https://wenruijiang1.github.io/) which facilitates interpolation, Lagrangian particle simulation, and Lagrangian tracer budget calculations. I helped write some [documentation]([/assets/images/seaduck_logo.png](https://macekuailv.github.io/seaduck/notebook/ASTE_sd.Particle_example.html)) for the package when it was expanded to work on the ASTE model grid. We also wrote a [tutorial](https://macekuailv.github.io/seaduck/bud_tut.html) to demonstrate the use of the Lagrangian tracer budget tool in Seaduck.
